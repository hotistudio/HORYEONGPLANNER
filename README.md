# 호령 플래너 (HORYEONG PLANNER)

> 인디게임 기획 올인원 관리 시스템

## 기능

- 🧑 캐릭터 · 스킬 · 아이템 DB
- 📜 퀘스트 구조화
- 🗺️ 세계관 (지역 · 세력 · 연표)
- 💬 대화 · 컷신 스크립트
- 📄 기획 문서 편집기
- 📅 마일스톤 · 개발 일정
- 🐛 버그 · 피드백 · TODO 트래커
- 🎵 사운드 에셋 관리
- 🎨 아트 에셋 관리
- 💰 수익화 · 손익분기 계산기
- 🎮 플레이테스트 세션 기록
- ⚖️ 수치 밸런스 계산기
- 🌙 다크 / ☀️ 라이트 모드
- 📱 모바일 · 태블릿 · 데스크탑 반응형
- ⚡ PWA (오프라인 지원)

## 사용법

GitHub Pages에서 바로 사용하거나, 로컬에서 실행:

```bash
npx serve .
```

> **주의:** Service Worker는 `file://` 프로토콜에서 작동하지 않습니다.  
> 반드시 로컬 서버 또는 GitHub Pages에서 실행하세요.

## 배포

`main` 브랜치에 push하면 GitHub Actions가 자동으로 Pages에 배포합니다.

GitHub 저장소 → Settings → Pages → Source: **GitHub Actions** 로 설정하세요.

## 데이터

모든 데이터는 브라우저 `localStorage`에 저장됩니다.  
우측 상단 **📤 내보내기** 버튼으로 JSON 백업이 가능합니다.
